## Description

Set up the codingagents pipeline to develop its own features, starting with ISS-001 (invariants-audit skill). This validates the framework's process against real development, surfaces usability issues, and establishes the practice of using the pipeline for ongoing framework work.

Design proposal: [docs/design/dogfood-proposal.md](../design/dogfood-proposal.md)

Key decisions:
