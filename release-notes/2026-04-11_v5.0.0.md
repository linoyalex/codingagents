## What's New

### Version 5 Baseline

`v5.0.0` establishes the `codingagents` v5 framework line carried by `version5-codex+token-governance` and its merge into `main`.

This baseline defines the core shape of the modern pipeline:

- **Structured handoffs** — `.claude/handoff.json` is the machine-readable contract between phases.
- **Token tracking** — `.claude/token-usage.jsonl` captures per-phase and per-agent usage.
- **Memory governance** — clearer separation between project memory, reusable skills, and handoff context.
- **Automated install and upgrade tooling** — `init.sh` and `upgrade.sh` become the primary deployment path.
- **Optional Codex review layer** — cross-model review is part of the v5 operating model.

`v5.0.0` is the generation baseline for all later `5.x` releases. Later minor and patch releases add stricter gates, new artifacts, and workflow refinements, but this release establishes the overall contract for the v5 pipeline line.
